## 1. 개요

### 1.1 목적

이 문서는 ITW(Infinite Text World)의 세계관과 서술 톤을 정의한다. 텍스트가 세계관 표현의 전부이므로, 이 문서는 두 가지 역할을 겸한다:

1. **세계관 레퍼런스** — 월드의 물리 법칙, 사회 구조, 분위기를 확립
2. **LLM 프롬프트 가이드** — 서술 생성 시 주입할 구체적 지침과 예문

### 1.2 적용 범위

| 적용 대상 | 방식 |
|-----------|------|
| 노드 진입/look 서술 | SensoryData + 오버레이 태그 + 도메인별 가이드 |
| NPC 대화 | ToneContext + attitude_tags + 대화 톤 가이드 |
| Echo/조사 결과 | Echo 카테고리 + 서술 톤 |
| 퀘스트 서술 | QuestOverlay narrative_tags + 톤 가이드 |

### 1.3 다언어 원칙

- **일본어 최우선**: Alpha 시 ja 필수
- 영어, 한국어: Beta 이후 순차 대응
- i18n 인프라는 Phase 2 후반 선투입, 하드코딩 방지

## 2. 세계관 기초

### 2.1 세계 정체성: "추방자들의 서역"

ITW의 무대는 **서역(西域)** — 동방의 대제국에서 추방된 자들이 살아가는 변경의 대지다.

| 속성 | 정의 |
|------|------|
| 장르 | 하이 판타지 + 자연철학. **건조한 경이(dry wonder)** |
| 핵심 전제 | 214개의 Divine Axiom이 이 세계의 물리 법칙 |
| 무대 | **서역(西域)** — 제국 추방자 + 원주민 이종족이 공존/대립하는 변경 |
| 톤 한마디 | "돌아갈 수 없는 자들이 만든, 마법 대신 공리가 있는 세계" |

"마법"이라는 단어는 쓰이지 않는다. 공리를 다루는 행위는 "기술"이자 "학문"이며 "직업"이다.

#### 서역의 위험성

서역은 제국이 "그냥 죽으라고" 추방자를 보내는 곳이다. 위험의 본질은 **땅 자체의 불안정성**:

- **바이옴의 비일관성**: 40% 확률로 인접 지역과 완전히 다른 기후/지형 출현
- **호전적 이종족**: 귀요정 부족이 서역 전역에 분포
- **야생 생물과 괴물**: 공리의 영향으로 변이한 생물체
- **공리 불안정 지대**: 미지의 공리 조합이 발현하는 위험 지역

> **LLM 프롬프트 발췌**
> ```
> 서역의 기본 분위기:
> - 안전한 곳은 마을과 도시뿐. 밖은 위험하다
> - 지형과 기후가 예측 불가
> - NPC는 마을 밖의 여행을 당연하게 위험하다고 인식
> ```

### 2.2 서역과 동쪽의 제국

#### 제국 — 맥거핀

제국은 동쪽에 존재하는(혹은 존재했던) 대국. **게임 내에서 실체가 드러나지 않는 맥거핀**.

알려진 것:
- 동쪽에 있다(있었다)
- 무차별적으로 인간을 추방한다(했다)
- 추방자의 몸에 "불가시의 자청"을 새긴다
- 화폐를 주조했다. 이 화폐가 만신전에서 통용된다

불확실한 것:
- 제국이 현재도 존재하는지 여부
- 근 10년 전부터 추방자가 기억을 잃은 채 도착하는 이유
- 제국의 구체적 모습, 문화, 정치 체계

> **설계 메모**: 제국의 실체는 의도적으로 미결정. NPC들의 추측과 소문으로만 존재.

#### 동쪽 경계

좌표 시스템에서 **x = 1이 동쪽 한계선**. **종족 불문 모두 사라진다.** 차이는 서술뿐:

| 대상 | 시스템 결과 | 서술 |
|------|-----------|------|
| 인간 (PC) | vanish (게임 오버) | 몸에 빛이 스쳤다. 그리고 쓰러졌다 |
| 인간 (NPC) | vanish (맵 제거) | "...동쪽으로 갔다더라" |
| 이종족 | vanish (맵 제거) | "경계를 넘어 걸어갔다. 돌아오지 않았다" |
| 동물 | vanish (맵 제거) | 동쪽으로 사라졌다 |

자청은 인간에게만 효과. 이종족이 사라지는 이유는 불명 — 경계 자체가 넘으면 돌아올 수 없는 문학적 장치. **일반 상황에서 어떤 NPC도 동쪽으로 이동하지 않는다.** 예외: 절망이 임계치를 넘은 인간 NPC.

```python
EASTERN_BOUNDARY_X = 1  # x=1이 동쪽 한계선

def check_eastern_boundary(entity, target_x):
    """x=0 이하로 이동 시 vanish. x=1은 최후의 관소(안전)."""
    if target_x <= 0:
        return "vanish"
    return None

def evaluate_eastern_march(npc):
    if npc.race != "human":
        return False
    if npc.emotions.get("despair", 0) > DESPAIR_THRESHOLD \
       and npc.emotions.get("hope", 0) < HOPE_MINIMUM:
        return True
    return False
```

#### PC의 시작

PC는 **기억을 잃은 상태로 동쪽 황야(x=1)에 버려져 있다.** 최후의 관소 주민이 수습.

- 자신이 왜 추방되었는지 모른다. 이름, 과거, 기술 — 전부 불명
- 근 10년 전부터 모든 추방자가 기억 상실 (PC만의 특수 상황이 아님)
- "나는 누구인가"가 개인적 메인 퀘스트의 씨앗

#### 관용어: "동쪽"과 "서쪽"

| 관용어 | 의미 | 사용 맥락 |
|--------|------|----------|
| "동쪽으로 갔다" | 죽었다 (완곡어법) | 사망 통보, 추모 |
| "서쪽 어딘가에 있지 않을까" | 살아있지만 행방불명 | 실종자 |

**인간만 자연스럽게 사용.** 이종족은 인용으로만:

```
인간: "미렌? ...작년에 동쪽으로 갔어."

소요정: "죽었지. 너희 추방자들은 뭐라고 하더라, 
'동쪽으로 갔다'? 그래, 동쪽으로 갔어."

숲요정: "인간은 죽음을 '동쪽'이라 부르지요. 
그 사람은... 네, 이 세상에 없어요."

귀요정: "죽었다. 인간의 말장난은 필요 없다."
```

> **LLM 프롬프트 발췌**
> ```
> 관용어: "동쪽으로 갔다"=사망, "서쪽 어딘가에"=행방불명
> 인간만 자연스럽게 사용. 이종족은 인용으로만.
> ```

#### 불가시의 자청(不可視の刺青)

- 육안 불가. 공리 감지 숙련자만 희미하게 감지
- 동쪽 경계 → 발동 → 즉사
- 자손에게 유전. 서역의 모든 인간이 보유
- 해제 방법 불명 (Mystery 도메인 퀘스트 소재)

#### 제국에 대한 NPC 태도

| NPC 배경 | 태도 | 대화 예시 |
|----------|------|----------|
| 직접 추방 (1세대, 희귀) | 분노/체념 | "...그 얘긴 하지 마. 50년 전 일이야." |
| 2세대 | 복잡, 정체성 갈등 | "부모님이 말해준 동쪽은... 정말 있긴 한 걸까." |
| 3세대 이후 | 무관심/전설 | "제국? 옛날이야기지. 여긴 여기야." |
| 최근 추방자 (기억 상실) | 혼란, 공포 | PC와 동류. 아무것도 모른다 |
| 학자 | 학문적 관심 | "자청의 공리 조합이 흥미로운데..." |

### 2.3 Divine Axiom과 일상

**공리는 누구나 사용 가능, 기술 전달 가능.** 재능이 아닌 수련의 깊이가 차이. 사회는 "마법사 계급"이 아닌 **기술자 길드** 중심.

| 도메인 | 수 | 일상 발현 | 서술 키워드 |
|--------|---|----------|-----------|
| **Primordial** | 30 | 화로의 Ignis, 우물의 Aqua | 원초적, 감각적 |
| **Material** | 30 | 대장장이의 Ferrum, 건축의 Petra | 견고한, 실용적 |
| **Force** | 30 | 풍차의 Ventus, 전사의 Kinetic | 역동적, 긴장 |
| **Organic** | 30 | 약초사의 Vita, 해독의 Venenum | 유기적, 순환적 |
| **Mind** | 30 | 학자의 Cognito, 예언자의 Visio | 내면적, 미묘한 |
| **Logic** | 30 | 기계공의 Machina, 법관의 Ratio | 기하학적, 건조한 |
| **Social** | 20 | 영주의 Imperium, 상인의 Commercium | 관계적, 정치적 |
| **Mystery** | 14 | Tempus(시간), Fatum(운명) | 초월적, 경이로운 |

> **LLM 프롬프트 발췌**
> ```
> "마법" 금지. 공리는 물리 법칙. 누구나 수련으로 사용 가능.
> "화로의 Ignis가 강철을 붉게 달군다" (O)
> "마법의 불꽃이 마력을 불어넣는다" (X)
> ```

### 2.4 종족

#### 인간 (Human)

제국 추방자의 후손 또는 기억 상실 최근 추방자. 공리 적응 변이는 있으나 종족적으로 동일.

#### 소요정 (小妖精 / Dwarf)

| 항목 | 설명 |
|------|------|
| 거주지 | 산악/지하. 인간 도시에도 확률적 거주 |
| 외형 | 인간의 절반~2/3. 다부진 체격 |
| 성격 | **시끄럽고 호기심 강하며 애교 많다**. 기술/소재를 보면 못 참음 |
| 대인 태도 | 기술적 흥미가 경계심을 이긴다. 약속은 반드시 지킬 것을 요구 |
| **톤 의도** | **서역의 어둠 속 의도된 대비.** "이 세계에도 순수한 호기심과 기쁨이 존재한다"는 증거 |

#### 숲요정 (森妖精 / Elf)

| 항목 | 설명 |
|------|------|
| 거주지 | 깊은 숲과 고원 |
| 수 | **극히 적다**. 마을 출현 자체가 드문 사건 |
| 성격 | **감정의 증감이 느리다**. 수백 년 수명. 인간의 시간 감각과 극심하게 어긋남 |
| 지식 | 기술, 역사, 지리, 약학 등 방대. 높은 공리 숙련 |
| 게임 역할 | **희귀 이벤트 NPC**. 세계관 떡밥 전달자, 퀘스트 와일드카드, 존재 자체가 "경이" |
| 특기 | 제국 이전, "돈 이전의 시대"를 기억하는 자가 존재 |

#### 귀요정 (鬼妖精)

**핵심: "모든 관계를 힘의 교환으로 이해하는 종족"**

| 항목 | 설명 |
|------|------|
| 거주지 | 전역. 인간 도시에도 확률적 거주(용병, 거래) |
| 성장 | 동일 종족. 인간 호칭: **소귀(小鬼)** 어린 개체 / **전귀(戦鬼)** 성체 / **대귀(大鬼)** 장로 |
| 식인 | 동족 포함 전방위. 죽은 자의 힘을 옮기는 의례. 최고의 경의 |
| 신용 | 약속 = 힘의 계약. 어기면 적. 한번 맺으면 목숨으로 지킴 |
| 장로 체계 | 가장 큰 자(= 가장 많은 힘 축적)가 지배. 허가 체계 |
| 돈 | 안 씀. 그들의 신은 돈을 안 받음. 약탈 시에도 돈만 남겨둠 |
| 거래 | **선물(贈物)제도** — 물물교환이 아닌 힘의 교환 의례 (하단 참조) |
| 과거 | 꽤 큰 국가를 건설한 전적 있음 |
| 대인 | 적대적이지만 무질서하지 않다. 경고→퇴거→전투. 힘의 논리를 이해하면 교섭 가능 |

```
종족간 인식 차이:

인간: "소귀 무리가 있었어. 전귀는 못 봤지만 있을지도."

소요정: "귀요정이 왔어! 아이들이면 장난감 보여주고 싶다!"

숲요정: "약속을 중시하는 종족이에요. 경계를 지키는 것뿐이지, 악의는 아닙니다."

귀요정(전귀): "경계선을 넘어오지 마라, 인간. 장로의 허가가 없다."
```

#### 이종족 간 관계

| 관계 | 내용 |
|------|------|
| 소요정 → 귀요정 | 껄끄러움(과거 전쟁). 절대적 적대는 아님 |
| 귀요정 → 소요정 | "바보지만 쓸모 있다". 거래 또는 노예화 |
| 숲요정 ↔ 모두 | 존경받는 스승 + 부탁 안 한 조언자(소크라테스) |
| 귀요정 → 숲요정 | 존경하지만 간섭이 귀찮다 |
| 전체 | 서로를 "그냥 있는 자들"로 인식. 서역의 원주민 |

> **LLM 프롬프트 발췌**
> ```
> 종족:
> - 인간: 추방자 후손. 다수
> - 소요정: 시끄럽고 기술광. 의도된 밝음(어둠 속 대비)
> - 숲요정: 극소수. 희귀 이벤트 NPC. 제국 이전을 기억
> - 귀요정: 힘의 교환이 모든 것의 기준
>   호칭: 소귀/전귀/대귀 (인간이 붙인 이름)
>   식인(의례적), 돈 안 씀, 신용 절대 중시
> - 인간 도시에도 소요정/귀요정 혼재 가능
> ```

### 2.5 기술 수준

**중세 후기 ~ 르네상스 초기**, 공리에 의한 비대칭 발전.

| 분야 | 수준 | 공리 영향 |
|------|------|----------|
| 야금/제조 | 르네상스 | Ferrum, Ignis로 현실 이상의 합금 |
| 의학 | 중세 말 | Vita, Venenum 숙련자 > 외과 |
| 건축 | 르네상스 | Petra, Forma로 현실 불가 구조물 |
| 교통/통신 | 중세 | Via, Lingua 숙련자 의존 |

화약 없음(Ignis+Kinetic 대체). 서역 기술 < 제국. 추방자의 단편 지식으로 재건 중.

### 2.6 경제와 화폐

#### 화폐의 원천

서역의 화폐 = **추방자가 가져오는 제국의 돈**. 조폐 능력 없음.

| 시기 | 상황 |
|------|------|
| 과거 | 대상인, 부호 등 추방. 대량 화폐 유입. 경제 활발 |
| 현재 | 기억 상실 추방자가 최소한의 짐만. 화폐 유입 격감. **경제 위기** |

#### "황제의 돈은 신에게까지 닿는다"

만신전의 사당은 **돈을 받으면 공리적 은총을 내린다. 언제나 일정하게.** 제국 화폐에 신과의 거래를 가능하게 하는 무언가가 있다.

함의:
- 추방 후에도 제국 화폐를 써야 한다 = **보이지 않는 지배**
- 돈 없으면 은총 없음 = **빈자의 절망**
- 경제 위기 = 신앙 위기

숲요정의 증언: "옛날에는 돈이 아니었어요. 들꽃 한 송이로도 Fons는 물을 내렸어요. 동쪽에서 온 사람들이 둥근 금속 조각을 가져오면서 바뀌었어요."

사당 운영자(사제) = **서비스업**. 누구나 돈 내면 은총. 사제에게 특별한 권위 없음.

#### Mercator — "피투성이 발바닥의 신"

**인간과 신 사이에 거래를 처음으로 체결한 존재.** 프로메테우스.

| 항목 | 설명 |
|------|------|
| 죄 | 신들의 거래를 지상으로 내린 것 |
| 벌 | 세상 어디에도 사당이 없다. 쉴 곳이 없다 |
| 봉납 | 먼 곳에서 온 상품을 구입하는 행위 자체 |
| 이미지 | 발바닥은 피투성이, 얼굴은 웃고 있다 |
| **시스템** | NPC 상점 재고 보충 시스템 = Mercator. "어디서 왔는지 모르지만 필요한 것을 파는 상인" |

#### 귀요정과 돈

돈을 안 쓴다. 그들의 신은 돈을 안 받는다. 약탈 시에도 돈만 남겨둠. 야만이 아니라 **신앙 체계가 다르기 때문**.

#### 귀요정의 선물(贈物)제도

귀요정은 화폐를 사용하지 않으므로, 물물교환도 아닌 **선물 제도**로 거래한다.

| 단계 | 내용 |
|------|------|
| 1. 제시 | 자신이 원하는 것을 밝히고, 그에 상응하는 물건을 상대 앞에 놓는다 |
| 2. 판단 | 상대가 놓인 물건을 보고 가치를 판단한다 |
| 3. 수락/거절 | 마음에 들면 물건을 받고 요구한 것을 준다. 아니면 돌려보낸다 |

- 값어치는 **귀요정 기준**(힘의 교환). currency 단위가 아님
- 무례한 제안(너무 적은 물건) = 모욕 → 관계 악화
- 적절한 제안 = 힘의 교환 성립 → 신용(약속) 형성
- **Alpha 후 구현 대상**. item-system.md 거래 시스템에 barter 모드 추가 필요

> **설계 메모**: 선물 제도는 귀요정 전용이지만, 화폐가 부족한 인간 NPC에게도 확장 가능 (경제 위기 연출)

> **LLM 프롬프트 발췌**
> ```
> 경제:
> - 화폐는 제국산. 사당에 내면 은총 (일정하게)
> - 현재 경제 위기: 추방자 빈손 → 화폐 유입 격감
> - Mercator: 사당 없는 신. 상품 구매가 봉납
> - 귀요정은 돈 안 씀 (신앙 체계 상이)
> ```

### 2.7 종교/신앙: 만신전(萬神殿)

#### 신격의 원리

| 계층 | 구성 | 신앙 방식 |
|------|------|----------|
| **단일 공리** | Ignis 등 개별 | 공경. 신앙 아님 |
| **소수 조합 (2~3)** | 순수한 신 | 깊은 경외. 엄숙한 제의 |
| **다수 조합 (4~6)** | 복합 신 | 일상적 기도, 축제 |
| **대규모 (7+)** | 태어나는 신 | 소규모 종파. 이단 취급도 |

#### 순수한 신 (2~3공리)

| 신격명 | 조합 | 영역 | 톤 |
|--------|------|------|-----|
| **Fornax** (화로신) | Ignis+Ferrum | 단조, 제련 | 엄숙, 뜨거운 |
| **Fons** (천수신) | Aqua+Terra | 샘, 관개 | 고요, 풍요 |
| **Mors Quieta** (안면신) | Necros+Pax | 죽음, 장례 | 차갑고 조용 |
| **Lucerna** (등화신) | Lux+Ventus | 길, 여행 | 따뜻하지만 쓸쓸 |

#### 복합 신 (4~6공리)

| 신격명 | 조합 | 영역 | 특기 |
|--------|------|------|------|
| **Aesculon** (치유신) | Vita+Cura+Sanguis+Antidotum | 의술 | 온화하지만 엄격 |
| **Mercator** (통상신) | Commercium+Valor+Via+Lingua | 교역 | **사당 없음** |
| **Custos Liminis** (경계신) | Auctoritas+Poena+Ferrum+Custodia | 법, 경계 | 근엄 |
| **Mater Segetum** (풍양신) | Terra+Fecunditas+Aqua+Vita+Tempestas | 농경 | 순환적 |
| **Silva Vox** (숲목신) | Vita+Flora+Tenebrae+Silentium | 숲, 야생 | 숲요정과 공유 |
| **Somniator** (몽견신) | Somnium+Memoria+Visio+Psyche | 꿈, 예언 | 흐릿, 불안 |
| **Pater Exilii** (유배신) | Poena+Crimen+Maledictum+Via | 추방, 속죄 | 서역 특유 |

**Pater Exilii**: "왜 우리는 여기에 있는가"가 이 신에 대한 기도. 축제 없는, 조용한 기도의 대상.

> **설계 메모**: Pater Exilii의 성장 → 동쪽 경계의 서진(후반부 위기 후보). "포기 섞인 신앙이 감옥을 좁힌다"는 역설.

#### 이종족의 신앙

| 종족 | 관계 |
|------|------|
| 소요정 | Fornax 열렬 공유. "우리가 먼저 모셨는데!" |
| 숲요정 | Silva Vox 공유. 더 오래된 이름으로 부름 |
| 귀요정 | 만신전 불인정. 독자 신앙(맹세와 의무, 힘의 교환) |

> **LLM 프롬프트 발췌**
> ```
> 만신전: 공리 조합 = 신격. 적을수록 오래된/순수한 신
> 사제 = 서비스업. 권위 없음. 돈 내면 은총
> Mercator: 사당 없음. 상품 구매 = 봉납
> Pater Exilii: 서역 특유. 추방 장면에서만
> 일상 속 자연스러운 신앙. 과도한 경건 금지
> ```

### 2.8 통치 구조

#### 최후의 관소(最後の関所)

서역의 유일한 입구. 동쪽 황야(x=1)의 중립 도시.

| 항목 | 설명 |
|------|------|
| 기능 | 추방자 수습. 화폐 유통 관문. 튜토리얼 도시 |
| 지위 | 절대적 중립지대 |
| 주민 | (아마도) 추방자가 아닌 제국민 후손. 자청 유무 검증 불가 — 시도하면 죽을 수 있어 아무도 안 함 |
| 위기 | 인구 자연감소. 쇠퇴 중 |
| PC | 기본 경험 후 "추방자는 오래 머물 수 없다"는 법으로 서쪽 추방 |

핵심 모순: 자기도 추방자일 수 있는데 추방자를 쫓아내는 법을 유지. 위선인가, 생존 전략인가, 집단적 자기 암시인가 — NPC마다 다른 답.

> **설계 메모**: 최후의 관소 상세는 별도 문서 `docs/20_design/starter-city.md`

#### 서역의 통치 유형

통일 정부 없음. 마을/도시마다 다르다.

**주요 (인간 정착지)**

| 유형 | 구조 | 길드 | 이종족 |
|------|------|------|--------|
| **영주제** | 추방 귀족(후손)이 지배. 제국식 위계 | 길드 회관 있으나 영주 아래 | 소요정, 귀요정 확률 혼재 |
| **자유도시** | 합의제/유력자 연합. 도시마다 법이 다름 | 길드 강한 발언권 | 동일 |

**기타 (저확률)**

| 유형 | 조건 |
|------|------|
| 무법촌 | 통치 부재. 힘 = 법 |
| 개인 독재 | 공리 숙련자 1인. 사망 시 붕괴 |
| 귀요정 보호령 | 인간이 공물 바치고 보호받음 |
| 폐허 점거 | 옛 유적에 거주 |

**이종족 고유 영역**

| 유형 | 특성 |
|------|------|
| 귀요정 부족령 | 장로 위계. 허가 필요 |
| 소요정 공방촌 | 기술 중심. 흥미 가져오면 환영 |
| 숲요정 영역 | 통치가 아닌 영역. 숲 = 그들의 공간 |

길드 = 도시 내부 직능 집단 (횡단 조직 아님). 척박한 서역에서 다양한 숙련자 협력이 생존 필수.

서역의 오래된 농담: "누군가가 Custos Liminis에게 빌었다, 조화를 달라고. 다음 해 법관과 영주가 추방당해왔고, 기도한 자는 농노가 되어 서역에 조화가 찾아왔다."

> **LLM 프롬프트 발췌**
> ```
> 통치: 서역에 통일 정부 없음. 마을마다 다름
> 영주제/자유도시가 주요. 이종족 영역도 있음
> 인간 도시에도 소요정/귀요정 혼재 가능
> 사제에게 정치적 권위 없음
> ```

## 3. 서술 톤앤매너

### 3.1 기본 톤: "건조한 경이(Dry Wonder)"

| 척도 | 위치 |
|------|------|
| 서사적 ↔ 일상적 | 중간-일상. 여행기에 가까움 |
| 다크 ↔ 밝음 | 중간. 어둠도 빛도 과장 안 함 |
| 유머 ↔ 진지 | 진지 쪽. 유머는 NPC(특히 소요정)를 통해 |
| 화려 ↔ 절제 | 절제 쪽. 명사와 동사로 장면을 그림 |

#### "건조"와 "경이"의 분리

"건조"는 **서술 톤**이 담당. "경이"는 **세계 자체의 현상**이 담당. 서술은 담담하지만, 묘사되는 현상이 이상한 것.

| 경이의 종류 | 원천 | Tier 매핑 |
|------------|------|----------|
| 경이 없음 | 일상 | Common |
| **규모의 경이** | 공리 발현의 스케일이 인간 상상을 넘음 | Uncommon |
| **조합의 경이** | 미관측 공리 조합 발생 | Rare |
| **Mystery의 경이** | 법칙 너머의 법칙 (시간, 인과, 존재) | 초레어 |
| **역사의 경이** | 숲요정의 증언, 유적, 제국의 흔적 | 이벤트성 |

```
경이의 예:

Common: "화로에서 Ignis가 타오른다. 대장장이가 망치를 든다."
→ 경이 없음. 일상.

Uncommon: "강이 역류하고 있다. 물은 맑고, 냄새는 보통 냇물과 같다."
→ 담담한 서술 + 이상한 현상 = 건조한 경이.

Rare: "돌이 노래한다. 낮고 일정한 진동. Petra와 Lingua의 조합인가?"
→ 서술은 관찰적. 현상이 경이.

초레어: "그림자가 3초 전의 당신을 비추고 있다."
→ Mystery 도메인. 법칙이 법칙답지 않은 순간.

이벤트: "숲요정이 말한다. '처음엔 들꽃으로도 Fons는 물을 내렸어요.'"
→ 과거가 지금과 이렇게 달랐다는 것. 시간 너머의 경이.
```

### 3.2 시점: 2인칭 현재형

| 언어 | 예시 |
|------|------|
| 한국어 | "당신은 먼지 낀 광장에 들어선다. 우물에서 물을 긷는 여인이 고개를 든다." |
| 日本語 | "あなたは埃をかぶった広場に足を踏み入れる。井戸で水を汲む女が顔を上げる。" |

예외: NPC 대화 = 1인칭(NPC 시점). 시스템 메시지 = 3인칭 무인칭.

### 3.3 문체 가이드라인

| 항목 | 규칙 |
|------|------|
| 문장 길이 | 10~25단어 (일본어: 15~40자) |
| 감각 묘사 | 전체의 40~60%. 시각 > 청각 > 촉각 > 후각 > 미각 |
| 비유 | 세계관 내적 비유만. 현대적 비유 금지 |
| 수식어 | 문장당 형용사/부사 최대 2개 |
| 동사 | 구체적 우선. "간다" < "걸어간다" < "성큼성큼 걸어간다" |

**좋은 예 / 나쁜 예**:

```
좋은 예:
"광산 입구에서 Ferrum의 냄새가 올라온다. 벽면의 철광석이 
횃불 빛에 어둡게 빛난다. 어딘가에서 곡괭이 소리가 울린다."

나쁜 예:
"마법적인 에너지로 가득한 신비로운 광산이 눈앞에 펼쳐진다! 
벽면은 아름답게 반짝이는 보석으로 장식되어 있으며, 
모험이 당신을 기다리고 있다!"
```

> **LLM 프롬프트 발췌**
> ```
> 서술 톤:
> - 시점: 2인칭 현재형
> - 톤: 담담하고 감각적. 과장 금지
> - "건조"는 서술이, "경이"는 현상이 담당
> - 감각: 시각+1개 이상 반드시 포함
> - 금지: 감탄사(!), 과도한 수식어, 현대적 비유, "마법"
> ```

| 日本語 | 감각적, 여운 | 1.1~1.2x |

| 대장장이 | Ferrum, Ignis 비유 | 금속, 도구, 날씨 | 학문, 정치 |
| 상인 | 가격, 거래 용어 | 시세, 도로, 소문 | 전투, 공리 이론 |
| 학자 | 정확한 용어 | 공리 체계, 역사 | 시세, 실전 |
| 경비병 | 간결, 명령조 | 치안, 위협 | 학문, 상거래 |
| 여관주인 | 친근, 일상적 | 음식, 손님, 소문 | 전문 분야 |

### 4.4 관계 상태별 톤 (여관주인, X고/A고)

| 상태 | 대사 |
|------|------|
| **Stranger** | "어서 오시게. 빈 방이 있소." |
| **Acquaintance** | "아, 자네로군. 이번에도 묵어가나?" |
| **Friend** | "왔구만! 자네 방은 비워뒀어. 스튜 한 그릇 하지?" |
| **Bonded** | "기다렸어. (낮은 목소리) 이상한 놈들이 마을에 왔더라고." |
| **Rival** | "...뭐 하러 왔소? 방은 다 찼소." |
| **Nemesis** | "나가. 자네한테 팔 방도 술도 없어." |

### 4.5 게임 정보의 자연스러운 전달

```
좋은 예: "그 풀이라면... 미렌이 알 거야. 북쪽 숲에 약초사 
오두막이 있거든. 다만 요즘 숲이 좀 으스스하다더라."

나쁜 예: "북쪽 좌표 (3,5)에 치유 약초가 있습니다. 수량 20개, 난이도 2."
```

> **LLM 프롬프트 발췌**
> ```
> NPC 대화:
> - manner_tags 따라 말투 조절
> - 종족 특유 말투:
>   소요정 = 활기차고 수다, 기술 흥분
>   숲요정 = 느리고 다정, 시간 어긋남
>   귀요정 = 짧고 무겁고 위엄, 허가/약속
> - 시스템 용어 절대 금지
> - 관용어: "동쪽으로 갔다"=사망. 인간만 사용, 이종족은 인용만
> ```

## 5. 맵/환경 묘사 톤

### 5.1 도메인별 감각 묘사

감각은 반드시 2개 이상. 도메인이 감각 종류를 결정.

#### Primordial (원초적)
```
"열기가 피부를 때린다. 균열 사이로 붉은 빛. 유황 냄새가 걸쭉하다."
```
키워드: 열, 냉기, 습기, 진동, 원소적 냄새.

#### Material (견고)
```
"돌벽이 좁아진다. 거친 화강암의 결. 건조한 먼지 냄새."
```
키워드: 질감, 무게감, 경도, 금속/광물 냄새.

#### Force (역동)
```
"바람이 협곡을 불어온다. 머리카락이 흩날린다. 낙석 소리."
```
키워드: 바람, 압력, 속도감. 문장도 짧고 빠르게.

#### Organic (생명)
```
"이끼 낀 나무들. 축축한 흙과 꽃향기. 벌레 소리가 쉬지 않는다."
```
키워드: 풀/나무/흙, 습기, 생물 소리.

#### Mind (내면)
```
"공기가 무겁다. 속삭이는 느낌이 의식 가장자리에 걸린다."
```
키워드: 압박감, 환청/환시 암시, 심리적 감각.

#### Logic (기계적)
```
"바닥 타일이 정확한 육각형. 벽면 문양은 수학적 규칙성. 딸깍 소리가 일정 간격."
```
키워드: 대칭, 패턴, 반복음, 정밀함.

#### Social (관계)
```
"광장에 사람들. 웅성거리는 소리. 화로 연기와 구운 고기 냄새."
```
키워드: 소음, 군중, 생활감.

#### Mystery (초월)
```
"그림자가 광원과 반대로 드리운다. 설명 못 할 향기 — 맡아본 적 없지만 기억나는 냄새."
```
키워드: 모순적 감각, 시간 왜곡, 기시감.

### 5.2 Tier별 분위기

| Tier | 톤 | 강도 | 길이 |
|------|-----|------|------|
| **Common** | 일상적 | 평범 | 2~3문장 |
| **Uncommon** | 이질적 | 약간 강함 | 3~4문장 |
| **Rare** | 경이/불길 | 강렬 | 4~5문장 |

### 5.3 날씨/시간

| 시간대 | 키워드 |
|--------|--------|
| Dawn | 안개, 이슬, 차가운 공기 |
| Morning | 햇살, 짧은 그림자, 활기 |
| Afternoon | 뜨거운 공기, 나른함 |
| Evening | 노을, 긴 그림자 |
| Night | 어둠, 별, 횃불, 고요 |

날씨 × 도메인 변주:
- 비 + Organic → "빗방울이 이파리를 타고 흘러내린다."
- 비 + Logic → "빗물이 타일 홈을 따라 정확하게 흘러간다."
- 안개 + Mystery → "안개가 발목에서 올라오지 않는다. 위에서 내려온다."

### 5.4 서브그리드 유형별 톤

| 유형 | 톤 | 감각 | 시간감 |
|------|-----|------|--------|
| 던전(하강) | 압박 | 어둠, 습기, 반향 | 사라짐 |
| 탑(상승) | 탁 트임 | 바람, 시야 확장 | 세계가 작아짐 |
| 숲(수평) | 방향 상실 | 식물, 벌레, 빛 차단 | 끊임없는 현재 |
| 동굴(3D) | 놀라움 | 물소리, 광물 냄새 | 지질학적 시간 |

> **LLM 프롬프트 발췌**
> ```
> 환경: 감각 2개+, 도메인 맞는 감각, tier별 강도 조절
> ```

## 6. LLM 프롬프트 가이드라인

### 6.1 필수 포함

| 요소 | 필수/권장 |
|------|----------|
| 감각 2개+ | 필수 |
| 도메인 힌트 | 필수 |
| 오버레이 반영 | 필수 (있을 때) |
| 인터랙션 암시 | 권장 |
| 시간대/날씨 | 권장 |

### 6.2 절대 금지

| 금지 | 이유 |
|------|------|
| 메타 발언 | 몰입 파괴 |
| 4th wall breaking | "플레이어님" |
| 시스템 용어 | "HP", "좌표", "난이도" |
| "마법" | 세계관 불일치 |
| 현대적 비유 | "전기가 흐르는 것 같은" |
| 감탄사 남발 | "놀랍다!" |
| NPC 내면 직접 서술 | 시점 위반 → 행동으로 |
| 플레이어 감정 지정 | "두려움을 느낀다" → "등줄기가 서늘하다" |
| 결과 예고 | 긴장감 파괴 |

### 6.3 길이

| 유형 | 한국어 | 日本語 | English |
|------|--------|--------|---------|
| look | 3~5문장 | 4~6문장 | 3~5문장 |
| move | 1~2문장 | 1~3문장 | 1~2문장 |
| investigate | 2~4문장 | 3~5문장 | 2~4문장 |
| NPC 대화 | 1~3대사 | 동일 | 동일 |
| 전투 | 2~3문장/라운드 | 2~4문장 | 2~3문장 |

### 6.4 SensoryData 활용

SensoryData = Python 시드. LLM이 확장.

규칙:
1. 시드 필드 직접 인용 금지
2. 영감으로 활용, 자연스럽게 재구성
3. 시드에 없는 감각 1개 이상 추가

## 7. META JSON과 서술의 이중 구조

모든 LLM 출력 = **서술(narrative) + META JSON**.

```json
{
  "narrative": "대장장이가 망치를 내려놓고 앞치마로 손을 닦는다. 'Fornax에 맹세하고 말하지만, 그 풀이라면 미렌이 알 거야.'",
  "meta": {
    "relationship_delta": { "affinity": 2, "reason": "helpful_information" },
    "memory_tags": ["asked_about_herb", "mentioned_miren"],
    "quest_hook": { "direction": "north", "poi": "herbalist_hut", "danger_hint": true },
    "tone_compliance": { "manner_tags_applied": ["verbose", "gentle"], "deity_referenced": "Fornax" }
  }
}
```

narrative = 플레이어 표시. meta = Python 엔진 소비.

## 9. 오버레이와 서술 톤 연동

### 9.1 narrative_tags → 서술

severity에 따라 묘사 강도 조절:

```
severity 0.3: "밭의 흙이 약간 마른 편이다."
severity 0.7: "땅이 갈라져 있다. 우물에 두레박을 내리면 바닥을 긁는다."
severity 1.0: "대지가 산산이 갈라져 있다. 우물은 말라붙어 바닥이 보인다."
```

### 9.2 퀘스트 오버레이의 NPC 영향

```
여관주인 (drought/despair): 
"비가 안 와. 석 달째. Fons에게 빌어봤지만... 
Mater Segetum도 우릴 잊은 건가."

농부 (drought/despair):
"(눈에 물기) 올해 수확은 틀렸어요. 씨앗이 전부..."
```

> **LLM 프롬프트 발췌**
> ```
> 오버레이: narrative_tags → 환경 묘사. severity로 강도 조절
> dialogue_tags → NPC가 자기 방식으로 상황 언급
> ```

## 10. 종합 LLM 프롬프트 템플릿

### 10.1 노드 진입 / look

```
[System]
あなたはITW（Infinite Text World）の語り手です。

世界設定:
- 舞台は「西域」— 東の帝国から追放された人間たちが暮らす辺境
- 214のDivine Axiomが物理法則として機能する世界
- 「魔法」禁止。「公理技術」と表現
- 種族: 人間（追放者の子孫）、小妖精、森妖精、鬼妖精
- 万神殿: 公理の組み合わせが神格を形成
- 経済危機: 追放者が記憶を失い手ぶらで到着。貨幣流入が激減

語りのルール:
- 二人称現在形（「あなたは〜する」）
- 感覚: 視覚＋もう1つ以上
- 「乾いた」は語りのトーン、「驚異」は世界の現象が担当
- 短い文。一文に一つの情報
- 禁止: 感嘆符, システム用語, 座標, 数値, メタ発言, 「魔法」

[User]
場所: {node_name}
支配Axiom: {dominant_axiom}
Tier: {tier}
時間帯: {time_of_day}
天気: {weather}
感覚データ: {sensory_data}
環境タグ: {narrative_tags}

3〜5文で描写を生成してください。
```

### 10.2 NPC 대화

```
[System]
あなたはITWのNPCとして会話します。

NPC情報:
- 名前: {npc_name}
- 種族: {race}
- 職業: {role}
- 性格要約: {hexaco_summary}
- 感情: {emotion} (強度 {intensity})
- 話し方タグ: {manner_tags}
- 態度タグ: {attitude_tags}
- 意図: {intent}

会話ルール:
- manner_tagsに従った話し方
- 職業/種族に合った語彙のみ
- 座標、数値、システム用語禁止
- 異種族は種族特有の話し方
- 慣用句: 「東へ行った」=死亡(人間のみ)、異種族は引用のみ
- 経済状況の自然な言及可能

[User]
状況: {situation}
環境タグ: {overlay_dialogue_tags}
PC発言: {player_input}

JSON形式で:
{
  "narrative": "台詞と動作描写",
  "meta": {
    "relationship_delta": {"affinity": 0, "reason": ""},
    "memory_tags": [],
    "quest_hook": null
  }
}
```

## 11. 구현 우선순위
