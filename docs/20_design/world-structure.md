# 월드 구조 설계

## 개요
1인용 텍스트 TRPG의 월드 구조

## 2단계 그리드 시스템

### 메인 그리드
- 좌표: (x, y)
- 무한 확장 가능
- 절차적 생성
- 이동: n, s, e, w (4방향)

### 서브 그리드
- 좌표: (sx, sy, sz)
- 메인 그리드 노드 내부
- 진입 조건 있음 (required_tags 등)
- 이동: n, s, e, w, up, down (6방향)

## 서브 그리드 유형
| 유형 | 구조 | 예시 |
|------|------|------|
| 던전 | 수직 하강 | 폐광, 지하묘지 |
| 탑 | 수직 상승 | 마법사 탑, 등대 |
| 숲 | 수평 미로 | 깊은 숲, 늪지 |
| 동굴 | 복합 3D | 자연 동굴 |

## 진입/탈출
- enter: 서브 그리드 진입
- exit: 서브 그리드 탈출 (입구로)
